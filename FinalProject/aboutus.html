About the Project

The P-VPN Method is a comparison and testing platform for VPNs, combining speed testing, leak testing, and protocol evaluation. It integrates technologies such as OpenVPN, WireGuard, and ProtonVPN into an interactive web application with local storage and a user-friendly interface.

My Story

My name is Price asemota from Co. Dublin and as a Computer Science student passionate about digital privacy, I built this platform to simplify VPN evaluation for users and develop my skills in full-stack web development and cybersecurity.
